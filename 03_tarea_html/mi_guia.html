Etiquetas de Formato

En desarollo Web, (HTML), las etiquetas "hr" y "b" se utilizan para dar formato basico
al contenido de una pagina. La etiqueta "hr" (Horizontal rule), inserta una linea
horizontal visible que sirve para separar visual o tematicamente diferentes secciones de texto.
Por otro lado, la etiqueta "b" (Bold), se utiliza para resaltar texto al hacerlo negrita,lo que ayuda a
resaltar visualmente palabras o frases especificas dentro de un parrafo.
